Vigil IoT

Vigil IoT is a desktop-first security platform for discovering local IoT devices, reviewing scan findings, and matching vulnerabilities with actionable remediation guidance.

View Repository See Screenshots

Project Introduction

The purpose of Vigil IoT is to help users understand device-level security risk across a local network. The system discovers reachable devices, identifies exposed services, and presents findings in a readable report format.

Major Feature 1: Device Discovery

Detects devices on a target network and prepares them for deeper analysis.

Major Feature 2: Service Analysis

Highlights exposed services and protocol-level security concerns.

Major Feature 3: Vulnerability Matching

Maps observed behaviors to known vulnerability categories for triage.

Major Feature 4: Report Management

Stores and retrieves scan results for future review and follow-up.
